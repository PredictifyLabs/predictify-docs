# Predictify Docs

Repositorio dedicado a la documentación oficial del proyecto.

## Contenido

- Arquitectura del sistema
- Diagramas de flujo
- Documentación tecnica de APIs
- Guia de despligue

## Estructura sugerida

docs/
├── database.md
├── architecture.md
├── backend.md
├── frontend.md
└── ai-module.md

### Descripción de archivos

- **[architecture.md](docs/architecture.md)**  
  Contiene la descripción detallada de la arquitectura del sistema, patrones utilizados y justificación de decisiones técnicas.

- **[backend.md](docs/backend.md)**  
  Documentación del servicio backend: estructura, módulos, dependencias, endpoints y lógica interna.

- **[frontend.md](docs/frontend.md)**  
  Explicación del funcionamiento del frontend, componentes principales, rutas y comunicación con el backend.

- **[ai-module.md](docs/ai-module.md)**  
  Incluye la documentación del módulo de inteligencia artificial, modelos utilizados, flujo de datos y procesos de inferencia.

- **[database.md](docs/database.md)**  
  Incluye la documentación de las entidades creadas dentro del sistema, sus atributos, relaciones y el diagrama entidad-relación (ERD) que representa la estructura de la base de datos.
